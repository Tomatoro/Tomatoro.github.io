### README

```bash
# 开启服务
hexo server
# 创建一篇文章
hexo new '文章标题'
# 清除缓存文件 (db.json) 和已生成的静态文件 (public)。
hexo clean
# 生成静态文件
hexo generate / g
# 保存自动更新
hexo generate -w
# 一键部署
hexo deploy / d
```
